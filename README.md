🌍 GlobeTrek AI – Full-Stack Travel & Booking PlatformGlobeTrek AI is a modern, high-performance travel agency platform built with the MERN Stack and Next.js. It leverages Google Gemini AI to provide intelligent destination suggestions and automated review summarization, offering a premium user experience for the modern traveler.
🚀 Key Features🤖
AI-Driven Experience
Gemini Intelligence: Integrated AI chatbot for personalized travel itineraries.
Smart Summaries: Automated sentiment analysis of user reviews using Generative AI.
Predictive Search: Advanced filtering based on user mood and budget.
🛡️ Enterprise-Grade Security
JWT Authentication: Secure login/registration with role-based access control (RBAC).
Protected Routes: Frontend and Backend middleware to separate User and Admin privileges.Bcrypt Encryption: Industry-standard password hashing.
📊 Administrative Suite
Real-time Analytics: Revenue tracking and user growth metrics via MongoDB Aggregations.
Inventory Management: Full CRUD operations for destinations and bookings.
Status Workflow: Manage booking lifecycles (Pending → Confirmed → Cancelled).
🛠️ Technical Stack
Frontend: Next.js, TypeScript, Tailwind CSS, Lucide React.
State Management: Zustand (with Persistent Middleware)
Backend: Node.js, Express.js, TypeScript.
Database: MongoDB, Mongoose (Aggregation Framework)AI EngineGoogle Generative AI (Gemini Pro)
API Client: Axios (with Interceptors for JWT)
